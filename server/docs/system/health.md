# 健康检查接口

## 接口信息

- **路径**: `/health`
- **方法**: `GET`
- **认证**: 不需要

## 请求头

无特殊要求

## 请求参数

无

## 请求示例

```bash
curl -X GET http://localhost:6808/health
```

## 响应格式

### 成功响应 (200)

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "status": "ok",
    "message": "服务运行正常"
  }
}
```

## 说明

- 该接口用于检查服务是否正常运行
- 不需要认证即可访问
- 可用于负载均衡器健康检查或容器健康检查
- 服务正常时始终返回 200 状态码
